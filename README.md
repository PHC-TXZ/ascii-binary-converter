# ASCII to Binary & Hex Representation 

A minimalist, responsive web application that provides real-time encoding data for characters. This tool is designed for developers and students to quickly inspect character codes.

## Features
* **Real-time Updates:** Results appear instantly as you type.
* **Multiple Formats:** Shows Decimal (ASCII), 8-bit Binary, and Hexadecimal.
* **Clean UI:** Modern design using Poppins typography and a mobile-friendly layout.
* **Accessible:** Built with semantic HTML and ARIA labels.

## How to Run
Simply open `index.html` in any modern web browser. No installation or build process is required.

## Tech Stack
* HTML5
* CSS3 (Custom Properties & Flexbox)
* Vanilla JavaScript
